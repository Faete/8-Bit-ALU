# 8 Bit ALU

  * 0000  :  ADD
  * 0001  :  SUBTRACT
  * 0010  :  MULTIPLY
  * 0011  :  COMPARE
  * 0100  :  SHIFT RIGHT
  * 0101  :  SHIFT LEFT
  * 0110  :  ROTATE RIGHT
  * 0111  :  NOT
  * 1000  :  AND
  * 1001  :  OR
  * 1010  :  XOR
